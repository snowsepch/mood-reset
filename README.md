# 情緒重開機

門市第一線人員專用的紓壓小工具。遇到奧客、退換貨爭議、業績壓力、系統當機等等煩心事時，用來快速冷靜、會心一笑，再回去工作崗位。

## 用途

- 選一個煩心事分類（或直接打字），按「碎紙釋放」，畫面把文字碎成紙片飛散消失。
- 碎完後跳出一句幽默又溫暖的回應句。
- 可選擇「深呼吸10秒再回崗位」，跟著圈圈做三輪 4秒吸氣、2秒停住、6秒吐氣。
- 完全不存資料、不連網路、不傳送任何內容，關掉頁面就什麼都沒留下。只用 localStorage 記今天用了幾次（純自己裝置本機計數，不含任何可識別內容）。

## 操作方式

雙擊 `index.html` 用瀏覽器打開即可，手機、電腦都能用（手機優先設計，也支援桌面版）。不需安裝、不需帳號、不需網路（唯一的網路請求是載入 Phosphor 圖示字型，離線時圖示可能不顯示但功能仍正常）。
