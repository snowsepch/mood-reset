# 情緒重開機

門市第一線人員專用的紓壓小工具。遇到奧客、退換貨爭議、業績壓力、系統當機等等煩心事時，用來快速冷靜、發洩壓力，再回去工作崗位。

GitHub Pages：https://snowsepch.github.io/mood-reset/

## 用途

- 選一個煩心事分類（或直接打字／語音輸入），按「碎紙釋放」，畫面把文字碎成紙片飛散消失，跳出一句幽默又溫暖的回應句。
- 不想選也不想打字，可以直接「用力打爆它」：打一顆發光星球，累積到一定拳數會爆炸釋放。
- 打完後可選擇「深呼吸10秒再去找財神爺」，跟著圈圈做三輪 4秒吸氣、2秒停住、6秒吐氣。
- 完全不存資料、不連網路、不傳送任何內容，關掉頁面就什麼都沒留下。只用 localStorage 記今天用了幾次（純自己裝置本機計數，不含任何可識別內容）。

## 操作方式

雙擊 `index.html` 用瀏覽器打開即可，或直接用手機開上面的 GitHub Pages 網址。畫面一律顯示在一個手機外框(iPhone)插畫裡（不管用電腦或手機開都一樣），真正可操作的工具在外框裡的螢幕區域，會自動縮放配合視窗大小。不需安裝、不需帳號（唯一的網路請求是載入 Phosphor 圖示字型跟手機外框樣式CDN，離線時外框可能不顯示但功能仍正常，因為 iframe 裡的 app.html 本身完全離線可用）。

## 路徑

- 正本：`iCloud:情緒重開機/`
  - `index.html`：手機外框包裝頁，實際入口（現在打開這個檔案，畫面就是一支手機插畫，裡面跑著真正的工具）
  - `app.html`：真正的工具本體，被 `index.html` 用 `<iframe>` 嵌進手機外框的螢幕裡；如果只想要全螢幕不要外框（例如要嵌進別的頁面），直接開這個檔案
- 公開網址：https://snowsepch.github.io/mood-reset/（repo：snowsepch/mood-reset，public，內容全為通用文案，無門市/客戶真實資料）
