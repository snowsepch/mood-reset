# 情緒重開機

門市第一線人員專用的紓壓小工具。遇到奧客、退換貨爭議、業績壓力、系統當機等等煩心事時，用來快速冷靜、發洩壓力，再回去工作崗位。

GitHub Pages：https://snowsepch.github.io/mood-reset/

## 用途

- 選一個煩心事分類（或直接打字／語音輸入），按「碎紙釋放」，畫面把文字碎成紙片飛散消失，跳出一句幽默又溫暖的回應句。
- 不想選也不想打字，可以直接「用力打爆它」：打一顆發光星球，累積到一定拳數會爆炸釋放。
- 打完後可選擇「深呼吸10秒再去找財神爺」，跟著圈圈做三輪 4秒吸氣、2秒停住、6秒吐氣。
- 完全不存資料、不連網路、不傳送任何內容，關掉頁面就什麼都沒留下。只用 localStorage 記今天用了幾次（純自己裝置本機計數，不含任何可識別內容）。

## 操作方式

雙擊 `index.html` 用瀏覽器打開即可，或直接用手機開上面的 GitHub Pages 網址。**手機上直接全螢幕使用工具，跟原來一樣**；只有在寬螢幕電腦瀏覽器打開時，才會自動套上一個手機外框(iPhone)插畫效果（純裝飾，方便桌面預覽/展示）。這個判斷是自動的，不用手動切換。不需安裝、不需帳號（唯一的網路請求是載入 Phosphor 圖示字型，電腦版外框額外會載入 devices.css 樣式；離線時圖示/外框可能不顯示但功能仍正常）。

## 版本

v1.3（2026-07-04 建立，同日上線 GitHub Pages）。手機外框曾經試過「不管裝置都套外框」，但這個做法在 iOS Safari 上會讓觸控完全失效（iframe/CSS transform 相關的已知瀏覽器限制），試了三種技術方案都無法穩定運作，最後改成「只有寬螢幕+滑鼠裝置才套外框，手機一律直接全螢幕」，靠偵測 `(min-width: 880px)` 加 `(pointer: fine)` 判斷，手機完全不會跑到套外框那段程式碼，保證跟原本一樣穩定。此工具不含門市名稱等客製內容，任何門市都可以直接共用同一份檔案，不需要重新產生專屬版本。
